How It Works

  1. Generate a token above (single-use or batch).
  2. Put the token in the device's config — add "registration_token": "TOKEN" to /boot/agora-config.json when flashing the SD card.
  3. Device connects using the registration token. It appears as Pending in the Devices page.
  4. Approve the device — the CMS generates a unique device auth token and pushes it to the device.
  5. On reconnect, the device uses its device auth token (not the registration token).
